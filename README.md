# Jianbing-Master
This project is an interactive jianbing customisation program written in Python.
## Project Overview
This project is an interactive jianbing customisation program written in Python. It allows users to create their own jianbing by choosing different ingredients, such as eggs, sauces, meat, crispy fillings, vegetables, and seasonings. After the user finishes selecting ingredients, the program calculates the total price, generates a final recipe, prints an order receipt, and displays the final jianbing with ASCII art.
The project is designed for students and young people who enjoy street food and want to quickly customise a breakfast or lunch option based on their taste and budget. It can also be used as a fun ordering simulator or as an idea for a more engaging digital ordering system for food stalls or campus food carts.
## Files
### **main.py**
This is the main entry point of the program. It prints the welcome interface and controls the main program loop. Users should run this file to start the jianbing ordering system.
### **jianbing_robot.py**
This file contains the core logic class of the project. It is responsible for handling the ordering process, calculating prices, managing invalid inputs, generating the final order, and reading the ASCII art/text files.
### **ingredients.py**
This file stores the ingredient data and prices. It includes different options for eggs, sauces, meat, crispy fillings, vegetables, and seasonings.
### **.txt Files**
The text files store ASCII art and ingredient descriptions. These files are used to make the program more visual and interactive.
